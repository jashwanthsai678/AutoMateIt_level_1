# 🎯 Python Automation Learning Game

A .NET Web API application that teaches automation through Python coding challenges with multiple difficulty levels.

## 🚀 Features

- **Multi-level System**: Three levels with increasing difficulty
- **Manual Guessing**: Traditional number guessing interface
- **Automated Guessing**: Python code editor with execution
- **Time Complexity Analysis**: Analyzes code efficiency
- **Progress Tracking**: Tracks completion of levels
- **Real-time Execution**: Python code runs on the server

## 🏗️ Architecture

- **Backend**: .NET 8 Web API with C#
- **Python Execution**: IronPython for .NET
- **Frontend**: HTML5, CSS3, JavaScript
- **API Design**: RESTful endpoints
